# La llamada de la diosa – TFG (RPG Maker MZ)

Este repositorio contiene el código fuente, recursos y estructura del proyecto desarrollado para el Trabajo Final de Grado.  

Se trata de un RPG narrativo creado con **RPG Maker MZ**, que incluye:

- Escena inicial con selección de género y nombre del protagonista  
- Mapa principal del pueblo  
- Viviendas y edificios funcionales (casa del jugador, casa del abuelo, tienda, armería, etc.)  
- Primeros eventos narrativos que dan inicio a la historia  

En la sección **Releases** se incluye una versión compilada del juego y un archivo ZIP con todos los recursos utilizados.
